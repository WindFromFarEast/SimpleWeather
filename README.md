# SimpleWeather
Version 1.0

一款简洁的天气App

功能：

1、可以查看全国各个地区的天气情况

2、每日自动更换应用壁纸

3、后台自动更新

4、下拉刷新

5、耗电量小（这是当然的啦哈哈哈）

注意事项：

由于使用了免费的天气接口，因此每天4000次访问次数用尽后将无法再次更新天气数据；由于每日壁纸是自动从必应官网获取的，因此如果必应壁纸很丑的话，App界面就会很丑，我也没办法哈哈哈哈

用到的开源库：Glide、LitePal、GSON、OkHttp

UI设计规范：一部分UI设计用到了Material Design
